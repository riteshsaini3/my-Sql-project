# MySQL Python Database Project

This project demonstrates how to connect **Python** with **MySQL** using the `mysql-connector-python` library and perform basic database operations such as fetching records from a table.

It is designed for **beginners**, **students**, and **DevOps / backend learners** who want hands-on experience with database connectivity in real-world scenarios.

---

##  Features

- Connects Python application to MySQL database
- Executes SQL queries using cursor
- Fetches and displays records from database tables
- Uses clean and readable Python code
- Suitable for learning production-style DB access

---

 Technologies Used

- Python 3.x
- MySQL
- mysql-connector-python
- VS Code
